# Changelog

## 0.1.0

- Initial release.
- Adds plugin service to auto-start local timeline dashboard.
- Supports dynamic sequence rendering for user/main/subagent interactions.
- Supports `sessions_spawn` and `sessions_send` visualization with mechanism labels.
- Adds process-event toggle (`显示过程信息`) for internal execution events.
- Adds spawn completion fallback parsing to avoid missing child-task return visualization.
